Menu item information

Menu items provide a pathway for users to access various features or pages in the application. They should be clearly labeled and easily recognizable to enhance navigation. Each menu item can be customized with icons, text labels, and links to specific sections.

SQL

SQL (Structured Query Language) is used to manage and manipulate relational databases. It allows users to retrieve, update, and delete data within a structured format, making it essential for data-driven applications.

Use Insert/Update

The INSERT statement in SQL is used to add new records to a table, while the UPDATE statement modifies existing records. These commands are essential for keeping database records current and accurate.

Use InfoSQL

InfoSQL is a collection of SQL commands used specifically to retrieve metadata about the database, such as table structures, data types, and index information. This helps developers understand the database schema better.

Grid settings

Grid settings allow developers to control the layout and appearance of data displayed in grid format, such as a table or matrix. These settings include row and column alignment, cell padding, and font styles.

Field information

Fields represent individual pieces of data within a database table. Each field has a specific data type, such as text, number, or date, and plays a crucial role in structuring the information within the database.

Advanced settings

Advanced settings provide additional configuration options for experienced users. These settings often include options for database optimization, permissions, and other complex configurations.

Hide menu item in menu group

Access permissions
